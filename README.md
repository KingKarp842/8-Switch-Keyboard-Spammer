# 8-Switch-Keyboard-Spammer

The 8-Key Keyboard Spammer is a simple handheld fidget device that uses mechanical keyboard switches to create a fidget toy. The design is made of a 3D-printed top plate that holds the switches, a bottom plate that it screws into, and screws that hold the two pieces together. The layout is based on my custom macropad I built recently and uses eight mechanical switches arranged in the same way.

After building my macropad, I found myself constantly clicking the switches because of how satisfying they felt. Instead of putting unnecessary wear on my actual macropad, I decided to design a dedicated fidget clicker that I could carry around and use whenever I wanted.
